## Google Maps Embed key setup

The Google Maps Embed URL is now generated by the Supabase Edge Function `google-maps-embed-url`.

Before deploying:
- Define the Supabase secret `GOOGLE_MAPS_EMBED_API_KEY`.
- In Google Cloud Console, restrict the key to your production/dev domains.
- Restrict the key usage to **Maps Embed API** only.
- Configure quotas to prevent abuse.
